# 📦 Projeto 08 – Curva ABC de Estoque

## 📖 Sobre o Projeto

Este projeto tem como objetivo aplicar a metodologia **Curva ABC** para análise e classificação de produtos em estoque, identificando os itens mais relevantes financeiramente para a operação.

A Curva ABC é uma técnica amplamente utilizada na logística e gestão de estoques para priorizar produtos, otimizar recursos e melhorar a tomada de decisão.

---

## 🎯 Objetivos

- Classificar produtos em Classes A, B e C.
- Identificar os itens com maior impacto financeiro.
- Calcular percentuais individuais e acumulados.
- Construir um dashboard visual para análise gerencial.
- Aplicar conceitos de gestão de estoque e análise de dados.

## 📈 Dashboard

O dashboard foi desenvolvido para fornecer uma visão estratégica do estoque através das seguintes análises:

### 🥧 Distribuição ABC
Mostra a participação percentual das Classes A, B e C.

### 📊 Valor por Categoria
Apresenta o valor financeiro por categoria de produto.

### 📊 Valor por Classe ABC
Compara o valor total representado pelas Classes A, B e C.

### 📉 Curva ABC
Exibe o percentual acumulado dos produtos ordenados por importância financeira.

---

## 📋 Metodologia ABC

### Classe A
- Produtos mais importantes.
- Representam aproximadamente 80% do valor acumulado.
- Necessitam maior controle e monitoramento.

### Classe B
- Produtos de importância intermediária.
- Representam aproximadamente 15% do valor acumulado.

### Classe C
- Produtos de menor relevância financeira.
- Representam aproximadamente 5% do valor acumulado.
