# 📊 Dashboard KPIs Logísticos

Projeto desenvolvido em **Google Sheets** com foco na criação de um **Dashboard Logístico** para monitoramento de indicadores de desempenho (KPIs), utilizando **Tabelas Dinâmicas**, **gráficos** e **indicadores estratégicos** para apoiar a tomada de decisão.

---

## 🎯 Objetivo

Desenvolver um painel executivo capaz de acompanhar os principais indicadores logísticos de uma operação, permitindo uma análise rápida sobre o desempenho das entregas, custos de frete e distribuição por cidade e transportadora.

---

## 📈 KPIs Desenvolvidos

- 📦 Total de Pedidos
- ✅ Total de Entregas Realizadas
- ❌ Total de Entregas Atrasadas
- 🚛 Pedidos em Trânsito
- 💰 Custo Total dos Fretes

---

## 📊 Dashboard

O painel apresenta visualizações gerenciais através de gráficos e indicadores:

- 🍕 Status das Entregas
- 📊 Custo por Transportadora
- 📊 Pedidos por Cidade
- 📈 Evolução das Entregas por Data

Todos os gráficos são alimentados por **Tabelas Dinâmicas**, proporcionando atualização automática conforme os dados são alterados.

## 💡 Principais Análises

- Quantidade total de pedidos processados
- Percentual de entregas realizadas
- Quantidade de entregas atrasadas
- Pedidos em trânsito
- Distribuição dos pedidos por cidade
- Participação das transportadoras no custo total
- Evolução das entregas ao longo do período
