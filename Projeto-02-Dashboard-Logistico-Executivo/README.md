# 📊 Dashboard Logístico Executivo

## 📌 Descrição

Este projeto foi desenvolvido utilizando **Google Sheets** com o objetivo de transformar dados operacionais em informações estratégicas para apoio à tomada de decisão.

O dashboard apresenta indicadores logísticos e gráficos interativos que permitem acompanhar o desempenho das operações de transporte de forma rápida e visual.

---

## 🎯 Objetivo

Construir um painel executivo para monitorar os principais indicadores logísticos de uma operação de transporte, facilitando a análise de desempenho e a identificação de oportunidades de melhoria.

---

## 🛠️ Ferramentas Utilizadas

- Google Sheets
- Tabelas Dinâmicas
- Gráficos Dinâmicos
- Fórmulas (CONT.SE, CONT.VALORES, SOMA)
- Análise de Dados

---

## 📈 Indicadores (KPIs)

- Total de Entregas
- Entregas Concluídas
- Entregas Atrasadas
- Entregas em Trânsito
- Custo Total dos Fretes

---

## 📊 Visualizações

O dashboard contém os seguintes gráficos:

- 🍕 Status das Entregas
- 📊 Custo de Frete por Transportadora
- 📊 Entregas por Cidade

---

## 🔍 Principais Resultados

- Total de Entregas: **30**
- Entregues: **20**
- Atrasadas: **5**
- Em Trânsito: **5**
- Custo Total de Fretes: **R$ 6.595,00**

---

## 💡 Insights Obtidos

- Aproximadamente **66,7% das entregas foram concluídas**.
- Foram identificadas **5 entregas atrasadas**, indicando necessidade de acompanhamento operacional.
- O dashboard permite identificar rapidamente custos logísticos e distribuição das entregas por cidade e transportadora.
