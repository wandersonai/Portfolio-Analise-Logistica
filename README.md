# 📊 Portfólio de Análise Logística e Dados

Bem-vindo ao meu portfólio de projetos!

Este repositório reúne projetos práticos desenvolvidos durante minha jornada de estudos em **Google Sheets, Excel, SQL e Power BI**, aplicando conceitos de **Análise de Dados, Business Intelligence e Logística**.

O objetivo é demonstrar minhas habilidades na transformação de dados em informações estratégicas para apoio à tomada de decisão, utilizando ferramentas amplamente empregadas no mercado.
