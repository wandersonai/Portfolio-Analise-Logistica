# 📊 Projeto 10 – Gestão Inteligente de Inventário

## 📌 Sobre o Projeto

Este projeto simula a gestão inteligente de inventário de um Centro de Distribuição (CD), utilizando um fluxo completo de Business Intelligence.

Os dados são armazenados em um banco MySQL, consultados através de SQL e transformados em um Dashboard Executivo no Power BI para apoiar a tomada de decisão.

O objetivo é monitorar o estoque, identificar produtos críticos, acompanhar movimentações de entrada e saída e analisar o valor financeiro do inventário.

---

## 🎯 Objetivos

- Controlar o estoque de produtos
- Monitorar movimentações de entrada e saída
- Identificar produtos em estoque crítico
- Calcular o valor financeiro do inventário
- Construir indicadores estratégicos (KPIs)
- Desenvolver um Dashboard Executivo para tomada de decisão

# 🗄 Banco de Dados

O projeto foi desenvolvido utilizando o banco de dados MySQL.

Foram criadas tabelas para representar:

- Produtos
- Categorias
- Fornecedores
- Movimentações de Estoque

Também foram desenvolvidas Views SQL para facilitar a construção dos indicadores utilizados no Power BI.

---

# 📈 Indicadores Desenvolvidos

O Dashboard apresenta indicadores importantes para a gestão do inventário:

- 💰 Valor Total em Estoque
- 📦 Produtos Cadastrados
- ⚠ Produtos Críticos
- 🏷 Categorias

Além disso, foram criados gráficos para análise de:

- Valor do estoque por categoria
- Produtos com estoque crítico
- Entradas x Saídas
- Evolução das movimentações
- Filtros por categoria e fornecedor
